📋 How to Set Up AnkiConnect

1
Install Anki — Download and install Anki on your computer. Make sure Anki is running when you want to create flashcards.
2
Install AnkiConnect add-on — In Anki, go to Tools → Add-ons → Get Add-ons, then paste the code: 2055492159 and click OK. Restart Anki.
3
Allow browser access — AnkiConnect listens on http://localhost:8765. You may need to allow CORS in AnkiConnect settings: Tools → Add-ons → AnkiConnect → Config. Add "webCorsOriginList": ["*"] to the config and restart Anki.
4
Create your deck — In Anki, click Create Deck at the bottom and name it (e.g. "Vocabulary"). Then enter the deck name above.
5
Test the connection — Click the Test Anki Connection button above. If it shows ✅, you're all set! If it shows ❌, make sure Anki is running and AnkiConnect is properly installed.
