Look up any word in context with dictionary, AI explanations, and Anki integration
=======

A modern Chrome Extension for language learners that provides interactive subtitles with dictionary lookup, AI explanations, and Anki integration for YouTube, Netflix, Amazon Prime, and standard web pages (including PDFs).

![Extension Screenshot](icon.png)

## Features

*   **Interactive Subtitles**: Hover and click on any word in a subtitle to get instant dictionary definitions.
*   **AI Context Analysis**: Get contextual translations, grammatical breakdowns, and nuances using external AI providers like **Mistral**, **Ollama** (Local), **OpenAI**, or **Gemini**.
*   **Anki Integration**: Automatically generate Cloze or Basic flashcards from your subtitle selections, pushing directly to your local Anki application.
*   **Universal Support**: Works on major streaming services (YouTube, Netflix, Amazon Prime) and allows generic right-click translation on any web page text or local PDF.

## Installation

### Prerequisites

*   Google Chrome or a Chromium-based browser (Edge, Brave).
*   (Optional but recommended) [Anki](https://apps.ankiweb.net/) desktop application with the [AnkiConnect](https://ankiweb.net/shared/info/2055492159) add-on installed.

### Loading Unpacked Extension

1.  Download or clone this repository:
    ```bash
    git clone https://github.com/YOUR-USERNAME/interactive-subtitle-dictionary.git
    cd interactive-subtitle-dictionary
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Build the project:
    ```bash
    npm run build
    ```
4.  Open your browser and navigate to `chrome://extensions/`.
5.  Enable **Developer mode** in the top right corner.
6.  Click **Load unpacked**.
7.  Select the `dist/` directory that was generated by the build process.

## AI Configuration

To use the AI translation features, you need an API key from one of the supported providers. Make sure to get your API Key and configure it in the Extension's Options page.

### Supported Providers

*   **Mistral AI** (Default, very fast)
*   **Ollama** (Free, private, runs locally on your machine)
*   **OpenAI** (High quality)
*   **Google Gemini** (High quality, generous free tier)

**(See `CONTRIBUTING.md` if you want to add support for a new provider!)**
